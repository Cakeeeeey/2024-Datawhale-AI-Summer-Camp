# Task 1 知识点
## 1.当前机器翻译技术
基于**神经网络**机器翻译（Neural Machine Translation，简称NMT）
NMT使用深度神经网络模型，如**长短期记忆网络（LSTM）和 Transformer**，能够自动学习源语言和目标语言之间的复杂映射关系，无需人工设计特征或规则。NMT在翻译质量、速度和适应性方面取得了显著进步，成为当前机器翻译领域的主流方法。
## 2.数据划分
- 训练集（Training Set）
- 测试集（Test Set）
- **开发集/验证集**（Development/Validation Set）
在模型训练过程中调整超参数、选择模型架构以及**防止过拟合**，用于评估模型在未见过的数据上的表现。
## 3.评估方法
### 3.1 BLUE来源
BLEU全称为Bilingual Evaluation Understudy（双语评估替换），由Kishore Papineni等人2002年的论文《BLEU: a Method for Automatic Evaluation of Machine Translation》中提出。
### 3.2 BLUE原理
衡量计算机生成的翻译与一组参考译文之间的**相似度**。这个指标特别关注 **n-grams**（连续的n个词）的精确匹配。

计算BLUE分数时，首先会统计生成文本中n-grams的频率，然后将这些频率与参考文本中的n-grams进行比较。如果生成的翻译中包含的n-grams与参考译文中出现的相同，则认为是匹配的。最终的BLUE分数是一个**介于0到1之间的数值**，其中1表示与参考译文完美匹配，而0则表示完全没有匹配。
### 3.3 BLUE特点
**BLUE** 评估指标的特点：

-   优点：计算速度快、计算成本低、容易理解、与具体语言无关、和人类给的评估高度相关。
    
-   缺点：不考虑语言表达（语法）上的准确性；测评精度会受常用词的干扰；短译句的测评精度有时会较高；没有考虑同义词或相似表达的情况，可能会导致合理翻译被否定。
